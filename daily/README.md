# 学習日記の使い方（GitHub運用）

このリポジトリは、毎日の学習内容（読書・技術学習・英語）を記録し、継続と振り返りをしやすくするための学習日記です。

---

## 1. 目的

- 学習を「見える化」して継続しやすくする
- 1日の実績を記録し、次の日の行動につなげる
- GitHub上に履歴を残し、学んだ内容を振り返れるようにする

---

## 2. 毎日使うテンプレート

以下を毎日の記録ファイルに貼り付けて使います。
